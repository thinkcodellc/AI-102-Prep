Azure AI Search can automatically extract text, identify key entities like names and dates, and make everything searchable in seconds. Users can search using natural language, not just keywords.

**Exam Focus**: Microsoft loves to test on skillsets and custom skills, understanding the difference between simple and full Lucene query syntax, and knowing when to use semantic search versus vector search for embeddings.
